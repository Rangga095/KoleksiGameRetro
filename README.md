# 🎮 Koleksi Game Retro (Java-MySQL CRUD)

Proyek sederhana berbasis Java untuk mengelola daftar koleksi game retro menggunakan database MySQL. Proyek ini mendemonstrasikan koneksi JDBC (Java Database Connectivity) dan operasi dasar CRUD.

## 🚀 Fitur
- **Koneksi Database**: Menghubungkan aplikasi Java ke MySQL Laragon.
- **Tambah Data**: Memasukkan koleksi game baru secara manual maupun interaktif.
- **Tampilkan Data**: Melihat daftar koleksi yang tersimpan dengan format tabel rapi di terminal.

## 🛠️ Tech Stack
- **Bahasa**: Java (JDK 25)
- **Database**: MySQL (via Laragon)
- **Library**: MySQL Connector J 9.5.0
- **IDE**: Visual Studio Code

## 📁 Struktur File
- `Koneksi.java`: Konfigurasi awal penghubung ke database.
- `TambahGame.java`: Menambahkan data game secara statis.
- `TambahGameInteraktif.java`: Menambahkan data game melalui input terminal.
- `LihatKoleksi.java`: Menampilkan seluruh isi tabel ke layar.
